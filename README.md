# WebService
- Server side for my Web Project

## Technology
- ASP.NET API
- Firebase Authentication
- SQL Server for database

## Deploy 
- Restore to get project dependencies
  ```bash
  dotnet restore
  ```
- Build project   
  ```bash
  dotnet build
  ```
- Service run at url <http://localhost:5234/api/>
  ```bash
  dotnet run
  ```
